02-11-2007 : V1.91

  • Correction d‘un bug de la largeur du fond splash sous IE7
  • Ajout de la gestion d‘autostart du diaporama 
      Il suffit de remplir les variables suivantes :
  - splash_auto_start = true si autostart activé sinon mettre à false
  - splash_auto_start_rank  = indice de l'image qui doit être lancée
  - splash_auto_start_group = par défaut est égale à 'splash' (pour les images non groupées), sinon mettre le nom du groupe de l'image

10-10-2007 : V1.9

  • pour les paresseux, il suffit désormais d‘ajouter "splash" au lieu de "splash.image" à l‘attribut "rel" (mais vous pouvez toujours conserver "splash.image" cela fonctionne encore)
  • Splash fonctionne désormais aussi avec les rel du type "lightbox" ou "lightbox[groupe]". Il est donc inutile de faire un masse update pour utiliser Splash (remarque private inside ^^).
  • Paramètrage du z-index du splash et de ses éléments pour éviter les problèmes de niveau des layers.
  • Cache et affiche les éléments de type embed ou object(flash entre autres) afin d’éviter que ceux ci passent par dessus le splash.
  • Une demande spéciale d‘un visiteur qui voulait afficher une image en fond à la place du fond noir. Pour cela il suffit dans le fichier splash.image.js de mettre le flag "set_bg" à "true" et de mettre dans le repertoire du script l‘image désirée en fond et de la renommer en "bg.gif".
